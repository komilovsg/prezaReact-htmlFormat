Что такое React

React — это библиотека для UI

  • ⚛️Создана в Facebook в 2013 году
  • 🧩Интерфейс = дерево компонентов
  • 🔄UI = f(state) — меняется стейт, UI обновляется сам
  • 💼Используется в Airbnb, Netflix, Yora и тысячах компаний
  • 📦Самая популярная JS-библиотека в мире (npm: 50M загрузок/нед)

Структура проекта

my-app/ ├── node_modules/ ├── public/ │ └── vite.svg ├── src/ │ ├── App.jsx ← главный компонент │ ├── main.jsx ← точка входа │ └── App.css ├── index.html ├── package.json └── vite.config.js
  • 📄 main.jsx — монтирует React в #root в index.html
  • ⚛️ App.jsx — первый компонент. Отсюда растёт дерево
  • 📦 node_modules — зависимости. Не трогаем, не коммитим
  • ⚙️ vite.config.js — конфиг сборщика. Пока не важен
Главная магия React

JSX — HTML внутри JavaScript

❌ Без JSX (больно)

React.createElement( 'div', { className: 'card' }, React.createElement( 'h1', null, 'Привет' ) )

✅ С JSX (красиво)

function Card() { return ( <div className="card"> <h1>Привет</h1> </div> ) }
Правило 1: class → className
Правило 2: один корневой тег или <>…</>
Правило 3: JS в JSX заворачивается в { }
Пишем живой код

Первый компонент

// src/App.jsx function App() { const name = "Шахрон" const role = "Frontend Developer" return ( <div className="card"> <h1>{name}</h1> <p>{role}</p> </div> ) } export default App
