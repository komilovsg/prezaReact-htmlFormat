Стандарт проекта

Папка src/components/

src/ ├── components/ ← новая папка │   ├── Button.jsx │   ├── Card.jsx │   └── UserCard.jsx ├── App.jsx ├── App.css └── main.jsx

export и import — обязательно знать

function Button() { return <button>Click me</button> } export default Button // ↑ без этого компонент невидим снаружи!

src/App.jsx

import Button from './components/Button' // ↑ путь от текущего файла function App() { return ( <> <Button /> <Button /> <Button /> </> ) }

Решение

Props — аргументы для компонента

Сначала без React — обычный JavaScript:

// функция с аргументом function sayHello(name) { return "Привет, " + name } sayHello("Ali") // "Привет, Ali" sayHello("Sara") // "Привет, Sara" sayHello("John") // "Привет, John"

В React — точно та же идея:

Вызов
<Card name="iPhone" />
→
Компонент получает
{ name: "iPhone" }

Props = аргументы, только записанные как HTML-атрибуты

Props в коде

Передаём данные через props

Card.jsx — получаем props

function Card(props) { return ( <div> <h2>{props.name}</h2> <p>{props.price}</p> </div> ) }

App.jsx — передаём разные данные

<Card name="iPhone 16" price="$1200" /> <Card name="Samsung S24" price="$900" /> <Card name="MacBook" price="$2500" />

Улучшаем код

Деструктуризация props

Это обычный JavaScript — React здесь вообще ни при чём

❌ Было (многословно)

function Card(props) { return ( <div> <h2>{props.name}</h2> <p>{props.price}</p> </div> ) }

✅ Стало (чисто)

function Card({ name, price }) { return ( <div> <h2>{name}</h2> <p>{price}</p> </div> ) }
Как это работает: функция получает объект { name, price } и сразу распаковывает его в переменные
Практика 1 — 20 минут

Компонент UserCard

Что нужно сделать:

// components/UserCard.jsx function UserCard({ name, age, job }) { return ( <div className="user-card"> <h2>{name}</h2> <p>Возраст: {age}</p> <p>{job}</p> </div> ) } export default UserCard

Как использовать в App.jsx:

<UserCard name="Ali" age={22} job="Frontend" /> <UserCard name="Sara" age={19} job="Backend" /> <UserCard name="John" age={30} job="Designer" />

⚠️ Числа передаются в {}, строки — в " "

Практика 2 — 20 минут

Компонент ProductCard

function ProductCard({ title, price, image }) { return ( <div className="product-card"> <img src={image} alt={title} width="200" /> <h3>{title}</h3> <p>{price}</p> </div> ) } export default ProductCard

Для картинок используй picsum.photos — бесплатные заглушки:

// Картинка 300×200 пикселей https://picsum.photos/300/200 // Конкретная картинка по ID https://picsum.photos/id/10/300/200 // Использование: <ProductCard title="MacBook" price="$2500" image="https://picsum.photos/300/200" />

Не тратьте время на поиск картинок — picsum даёт случайное фото по ссылке
